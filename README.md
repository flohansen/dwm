# My beautiful dwm
This is my personalized version of [dwm](https://git.suckless.org/dwm) including several changes to make the default dwm appearence more beautiful. If you want to use this project, you can. But please be aware about the fact, that most of the official patches of dwm will produce merge conflicts and may not work.

## Changes I've made
The following changes were made.

    - Add spacing between the clients
    - Add padding to the bar
    - Apply the official alpha patch
